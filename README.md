# BicycleGAN in `pytorch`
This an implementation of [Toward Multimodal Image-to-Image Translation](https://arxiv.org/abs/1711.11586).

## Requirements
1. pytorch 1.0
2. numpy, Pillow, opencv

## How to train
Just run `python train.py`.
